##### query

query is a json + `;`
example
```json
{
    "type": "insert",
    "values": [
      {
        "colName" :"columnA",
        "value": "fst val"
      },
      {
        "colName":"columnB",
        "value": 1
      }
    ]
  }
```
the other examples can be found at [query.json](query.json)
